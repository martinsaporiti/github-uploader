## How to use:

1. Register Application on Github
2. Create CLIENT_ID and CLIENT_SECRET env variables:

```bash
export CLIENT_ID=...
export CLIENT_SECRET=...
```

3. Run `go run main.go`
4. Visit `http://localhost:9000/`
